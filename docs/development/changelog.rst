Breaking Changes Policy
=======================

- All breaking changes happen in major versions (v2.0.0, v3.0.0, etc.)
- Deprecations are announced 1 major version in advance
- Minor versions (v1.1.0, v1.2.0, etc.) are always backward compatible
- Patch versions (v1.0.1, v1.0.2, etc.) fix bugs without changes
